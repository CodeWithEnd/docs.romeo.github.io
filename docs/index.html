Starting guide:

Command List

You can access the command list by executing help command

You can also use the reactions to navigate through categories

Logging

You can enable logging by executing togglelog command and providing a channel after the option. Once you do that, bot will start logging whatever you choose to your chosen channel. To disable logging, you execute the same command but without any arguments after the option

Automoderation

You can enable automod by executing -automod toggle command and then using automod command to choose what automod should look out for.

Music

You can now play music using Dredd, we also offer filters so you could change the pitch or speed of the music you're listening to.
